## What it is

A friendly tutor named Bloom that teaches reading, phonics, math, writing, science, social studies, and feelings — through bite-sized 60–90 second activities your kid actually wants to do.

Pick a world: 🦄 unicorns · 🧜‍♀️ mermaids · 🦕 dinosaurs · 🚀 space · 🐱 cats · 🐴 horses · 🐶 Bluey

## What makes it different

- 🌱 **Adapts** — gets harder as your kid gets stronger, drops back gently if they're struggling
- 🌟 **Click-only safety** — no microphone, no typing, no AI chat with strangers. Click the right answer, that's it.
- 🎉 **Real rewards** — sparkly stickers, full-screen confetti for streaks
- 🔇 **Works with sound off** — every answer shows a big visible green ✓ or red "Try again!"
- 💸 **Free forever** — no subscription, no upsell, no in-app purchases

## Scope

This is a personal project — a learning app I built for my daughter Jane and put online so other parents' kids can use it too. The hosted version above is the easiest way to try it. The repo is here mainly to show the project exists and to accept feedback.
